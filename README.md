# Разбор работы РОП — 11 контуров, июнь 2026

Один самодостаточный HTML: аналитика, цифры и оценка качества управления в одном документе.

## Что внутри

| Вкладка | Что показывает |
| :---- | :---- |
| **Обзор** | Внедрение недельного формата: 11 контуров, 3 РОПа, переход 4–8 июня 2026 |
| **Цифры май → июнь** | Короткая числовая сводка: деньги, активность, списки, нормативы, расхождения |
| **Качество управления** | Оценка работы РОПа как руководителя: июнь, дела Мегаплана + ABC |
| **Шокин / Переверзев / Огнева** | Развернутый разбор по каждому из 11 регионов |

## Источники

- 150 писем «Планы / Итоги» за январь — июль 2026, 10 email-веток;
- 8 975 дел Мегаплана за июнь по 11 контурам;
- ABC-выгрузка от 02.07.2026 — помесячная выручка по каждому клиенту.

## Главное из блока «Качество управления»

Формат работает: все три РОПа ведут разбор конкретных клиентов с причиной и следующим шагом, замыкают петлю поручений, фиксируют невыполнение.

Дыра — в охвате. Список на разбор собирается вручную, 10–20 клиентов на менеджера. В июне ниже своей базы просели **1090 клиентов категорий A и B**, и к **565 из них (51,8%)** не пришел никто: ни дела в Мегаплане, ни строки в письме. Падение по ним — **10 875 984 руб.**

Разрыв одинаков у всех трех РОПов (48–57%) — значит ограничение системное, а не личное.

## Как считано

- **Категория клиента** — по фактической выручке месяца: A ≥ 50 000 ₽, B 16 000–49 999 ₽, C < 16 000 ₽, D = 0. Правило из письма РОП, не кумулятивная категория из листа ABC.
- **Просевший** — категория июня ниже базы (лучший месяц из марта — мая). Разовое падение к маю не берется: регламент требует смотреть тренд.
- **Формальное закрытие** — у дела нет итога ни в комментарии, ни в теме. Без учета темы показатель завышается втрое.
- **Сезонность исключена**: июнь к маю по 11 контурам +2,2%.

## Ограничения

- В выгрузке Мегаплана нет поля «постановщик задачи» — отличить задачу РОПа от заведенной менеджером нельзя.
- Связка дела с клиентом ABC — только по точному совпадению имени (69% дел с указанным клиентом). Нечеткое сопоставление отброшено: подставляло однофамильцев.
- Связь управления с выручкой **не доказана**: коэффициент корреляции −0,02. Одного месяца и десяти точек для вывода недостаточно.

## Публикация

Файл самодостаточный, сборка не нужна. GitHub Pages: Settings → Pages → Deploy from branch, папка с `index.html`.

Ссылки на вкладки работают через якорь: `index.html#upravlenie`, `#cifry`, `#shokin`.

Единственная внешняя зависимость — шрифты Google Fonts (Space Grotesk, DM Sans, JetBrains Mono). Без сети страница отрендерится системным шрифтом, данные не пострадают.
